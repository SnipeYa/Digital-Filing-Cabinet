Digital Filing Cabinet

This application is intended for keeping track of household finances with all data being stored locally.

- Recieve Bills/Invoices via email and scrape the information for payment reminders and cost tracking
- Import bank transactions for vendor matching and expense categorization
- Create budgets and compare that with actual spending
- Keep track of loans with extra repayments and intrest saved

This is a desktop application that runs locally on the user’s computer. It uses the Gmail API only for the user’s own Gmail account. No data is sent to external servers.
